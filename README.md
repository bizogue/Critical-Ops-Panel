# 🛡️ Critical Ops Panel

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![License](https://img.shields.io/badge/license-MIT-green?style=for-the-badge)

Um painel de comando e controlo de alta performance desenvolvido em **Python (Tkinter)**, desenhado para centros de monitorização e resposta a incidentes críticos. Esta ferramenta permite que operadores acedam a protocolos de emergência de forma instantânea, com monitorização de tempo real.

---

## ✨ Funcionalidades Principais

| Funcionalidade | Descrição |
| :--- | :--- |
| 🚨 **22 Ações Rápidas** | Organizadas em 11 categorias (Energia, Segurança, Redes, etc). |
| ⏱️ **Cronómetro Dinâmico** | Monitoriza o tempo de execução com mudança visual de cores conforme a gravidade. |
| 🌗 **Dual Theme** | Alternância instantânea entre Modo Claro e Modo Escuro. |
| 📧 **Integração Outlook** | Abertura automática do e-mail para registo de ocorrências ao ativar um protocolo. |
| 📊 **Relatório de Tempo** | Feedback ao operador sobre o tempo exato gasto na resposta ao fechar o protocolo. |
| 🖥️ **Full-Screen UI** | Interface focada que elimina distrações externas durante uma crise. |

---

## 📸 Demonstração Visual

O sistema utiliza uma lógica de cores no cronómetro para pressionar a eficiência da resposta:
- **Verde**: 00:00 - 02:30 (Resposta dentro do tempo ideal)
- **Amarelo**: 02:30 - 04:00 (Atenção redobrada)
- **Laranja**: 04:00 - 05:00 (Limite de segurança)
- **Vermelho**: > 05:00 (Tempo de resposta excedido)

## 🛠️ Tecnologias Utilizadas

- **Tkinter**: Interface Gráfica (GUI).
